## 🚀 Running a Game Locally

Each game folder contains its own setup instructions and README.

General method:

```bash
# Clone the repository
git clone https://github.com/<your-username>/games.git

# Open a game folder
cd games/grid-masters

# Run with any local server
npx serve .
```

Or simply open the `index.html` file if the game supports direct execution.
